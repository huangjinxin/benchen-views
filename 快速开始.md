# 快速开始指南

## 一分钟上手

### 1️⃣ 启动后端服务器

```bash
cd backend
npm run start:dev
```

确保服务器运行在 `http://localhost:8891`

### 2️⃣ 打开网页

直接用浏览器打开：
- **填写记录**: http://localhost:8828/index.html
- **查看记录**: http://localhost:8828/records.html

### 3️⃣ 开始使用

系统会自动连接到服务器，无需任何登录操作！

---

## 工作原理

系统使用配置的账号（admin@beichen.com / admin123）自动在后台获取访问权限。

当你打开页面时，系统会：
1. 自动获取访问令牌
2. 自动加载园区、班级、教师数据
3. 准备好让你使用

**你只需要专注于填写和查看观察记录！**

## API端点

| 功能 | 端点 |
|------|------|
| 创建记录 | `POST /records/daily-observation` |
| 获取列表 | `GET /records/daily-observation` |
| 获取详情 | `GET /records/daily-observation/:id` |
| 更新记录 | `PUT /records/daily-observation/:id` |
| 删除记录 | `DELETE /records/daily-observation/:id` |

## 常见问题

**Q: 打开页面后没有反应？**
- 检查后端是否运行
- 打开浏览器控制台查看错误信息

**Q: 保存失败？**
- 确保填写了必填字段：日期、天气、老师姓名、班级
- 检查后端数据库中是否有对应的教师、班级数据

**Q: 如何修改后端地址？**
- 编辑 `api-config.js` 中的 `baseURL`

**Q: 如何修改认证账号？**
- 编辑 `api-config.js` 中的 `credentials`
